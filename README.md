# Rebindaine_et_al_2025_beech_phenology
This repository holds the code used to analyse bud length and leaf chlorophyll data from experiments 1 and 2 in the manuscript "Developmental constraints mediate the summer solstice reversal of climate effects on the autumn phenology of European beech" by Rebindaine et al. (2025) in eLife.
